# Machine Learning for Anomaly Detection in Virtualized Networks

## Overview
This project evaluates five machine learning algorithms (Random Forest, Gradient Boosting, Decision Tree, KNN, and ANN-MLP) for network intrusion detection using the CICIDS2017 dataset.

 key Findings
Random Forest achieved the best performance (99.80% accuracy, 99.99% AUC)
 Comprehensive feature engineering reduced dimensionality from 80 to 40 features
 SMOTE successfully addressed class imbalance in training data

 Technologies Used
Python programming data preprocessing and modeling
Machine Learning Algorithms:Random Forest, Gradient Boosting, Decision Tree, KNN, ANN-MLP
Dataset: CICIDS2017 (164,573 network traffic records)
